# 疯狂碰碰车 🚗💥

卡通暴力美学多人在线碰碰车大乱斗：一张持久大地图、无赛局、死亡重生、
高速对撞掉血击飞、氮气追逐、实时聊天、实时/历史双排行榜。
一个 URL 即可多客户端联机，支持 Docker 一键部署。

## 玩法

- **高速撞击**才会造成伤害：撞得越快，伤害和击飞越夸张（弹性系数 > 1，越撞越飞）
- 血量归零 → 车子**旋转着飞上天爆炸**（KO!），击杀者得 150 分，3 秒后原地图重生
- 得分是"这条命"的：死亡清零并计入**历史排行榜**（按名字保留最佳战绩，持久化到磁盘）
- 全场分数最高者头顶 **👑**——干掉国王得 **300 分**，全服追着国王跑
- **氮气冲刺**（有能量槽）：冲刺撞击伤害 +35%，是追逐与逃命的核心
- 地图机关：**弹球柱**（BOING 弹飞）、**电锯**（掉血 + 弹飞）、**加速带**
- 道具：🔧 回血 · ⚡ 氮气回满 · 🛡️ 6 秒护盾 · ⭐ 8 秒狂暴（双倍伤害/击飞）

## 操作

| 按键 | 功能 |
|---|---|
| WASD / 方向键 | 朝该方向行驶，支持斜向（车头自动转向）|
| 空格 / Shift | 追加氮气冲刺 |
| H | 按喇叭 📢 |
| 回车 | 聊天 |
| Tab（按住）| 查看历史排行榜 |
| M | 静音 |

手机浏览器自动启用虚拟摇杆 + 冲刺/喇叭按钮。

## 本地运行

```bash
npm install
npm start          # http://localhost:3000
```

```bash
npm test               # 物理/规则单元测试
node scripts/e2e.js    # 3 客户端联机冒烟测试（需先启动服务器）
node scripts/bots.js 5 # 放 5 个会追杀、会喷垃圾话的机器人进场
```

## Docker 部署

```bash
docker compose up -d --build
```

打开 `http://服务器IP:3000`，把链接发给朋友即可开撞。
历史排行榜保存在 `./data/leaderboard.json`（已挂卷，重启/重建容器不丢失）。

环境变量：

| 变量 | 默认 | 说明 |
|---|---|---|
| `PORT` | `3000` | 监听端口 |
| `DATA_DIR` | `/app/data`（Docker）/ `./data`（本地）| 排行榜持久化目录 |

## 架构

参考同仓库 `games/arena-brawl` 的联机架构：

- **同端口一个 URL**：Express 托管静态客户端，`ws` 在同一 HTTP 端口升级 WebSocket
- **服务端权威模拟**：30Hz 物理 tick（输入只收方向布尔值，无法作弊），15Hz 全量快照广播 + 一次性特效事件流
- **客户端插值渲染**：在最近两帧快照间 lerp，断线自动重连并重新加入
- **背压保护**：客户端缓冲积压超阈值时跳帧；心跳清理僵尸连接
- **排行榜双轨**：实时榜来自世界状态每秒推送；历史榜死亡/离线时落盘（防抖写入）

```
server/
  index.js       # HTTP + WebSocket 网关、消息路由、三大循环（模拟/广播/排行）
  world.js       # 物理世界：碰撞击飞、伤害判定、机关、道具、重生、皇冠
  leaderboard.js # 历史榜持久化（同名保留最高分）
  config.js      # 所有数值与地图布局
public/
  js/net.js      # WebSocket + 快照插值    js/render.js # Canvas 卡通渲染与特效
  js/input.js    # 键盘/触屏输入           js/hud.js    # 聊天/排行/播报/死亡界面
  js/audio.js    # WebAudio 合成音效       js/main.js   # 入口与游戏循环
```
